
# Programmable Resume

Helps maintain my resume as **Markdown** and generate PDF, Word, or HTML versions automatically.

## Requirements

Install Pandoc:
https://pandoc.org/installing.html

Install LaTeX engine (for PDF):
https://tug.org/mactex/ (Mac recommended)

## Build Resume

Run: bash build.sh

Outputs will appear in: output/

## Files

resume.md — Main resume content  
build.sh — Script to generate resume formats  
output/ — Generated resumes
